# 数据库(ORM)

- TiDB ->>
    - PingCAP 团队开发
    - 分布式 SQL 数据库
    - 传统 RDBMS + NoSQL   

- influxdb
    - 可以水平扩展的时间序列数据库
    - @code https://github.com/influxdata/influxdb

- cockroach
    - 可伸缩的、支持地理位置处理、支持事务处理

    - @code https://github.com/cockroachdb/cockroach

- cayley
    - Graph数据库
    - @code https://github.com/cayleygraph/cayley

- dgraph
    - Fast, Distributed Graph DB
    - 支持 GraphQL，响应 JSON
    - @code https://github.com/dgraph-io/dgraph    

- codis

    - @code https://github.com/CodisLabs/codis
